Html можно назвать внутреностями сайта. Тоесть html это ничего большего чем текст или сылки со списками. С html работать легче чем например с css о котором кстате будет написано дальше.
И так css мы уже не можем сравнить с внутреностями сайта поскольку css скорее всего кожа. Потому что css задает к примеру цвет шрифта, фон, размер шрифта и прочие. С css работать уже сложнее чем с html.
